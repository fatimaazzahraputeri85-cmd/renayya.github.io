Tentang Website

Website ini dibuat sebagai tugas mata kuliah Jaringan Komputer untuk memahami konsep web hosting menggunakan GitHub Pages dengan memanfaatkan teknologi HTML dan CSS.

Tujuan Pembuatan

  • Memahami konsep dasar web hosting
  • Mengimplementasikan website statis
  • Mengunggah website ke GitHub Pages

Teknologi yang Digunakan

  • HTML (struktur website)
  • CSS (tampilan website)
  • GitHub Pages (hosting)
